# Zrøven

Plugin store de Minecraft — landing "coming soon".

## Estructura

```
zroven/
├── public/
│   ├── index.html   ← página principal
│   ├── styles.css   ← estilos
│   └── main.js      ← lógica del formulario y animaciones
├── package.json
└── README.md
```

## Uso

```bash
npm install
npm start       # sirve en http://localhost:3000
npm run dev     # live-reload en desarrollo
```

## Tecnologías
- HTML5 semántico
- CSS custom properties (sin framework)
- Vanilla JS (sin dependencias de frontend)
- Google Fonts: Cinzel + Space Grotesk
- `serve` para producción / `live-server` para dev
